# Strategy Meeting

Run a structured brainstorming session with cognitively diverse agents.

## Steps

1. Check if `.crypto/BOOTSTRAP.md` exists. If not, run init first.

2. Read context:
   - `.crypto/BOOTSTRAP.md`
   - `.crypto/knowledge/registry.yaml`
   - `.crypto/knowledge/search-space-map.yaml`
   - `.crypto/config/strategy-meeting-protocol.yaml`

3. Run meeting phases (Diverge -> Collect -> Filter -> Select -> Learn)

4. Present results and queue selected hypotheses for L0 validation.

## Usage

- `/crypto:meeting` - Open brainstorming
- `/crypto:meeting altcoin strategies` - Themed meeting
- `/crypto:meeting continuous` - Loop until stopped
